```

https://www.123xxxx.com/ [✅]
http://www.122xxxxx.com/login.html [❌]

```



那么扫描的结果会放在哪里呢？

```
当前目录下的reps文件夹中，会以漏洞名称-时间来命名，如xxxSql-2023-11-30
```
